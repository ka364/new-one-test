## البنية المشتركة

### قاعدة البيانات
جميع الميزات تستخدم:
- MySQL/PostgreSQL للبيانات الأساسية
- Redis للتخزين المؤقت
- Drizzle ORM للتفاعل مع قاعدة البيانات

### التكامل مع KAIA
كل ميزة تتضمن قسم للتوافق مع نظام KAIA للامتثال الشرعي:
- التحقق من المنتجات
- التسعير العادل
- الشفافية

### APIs
جميع الـ APIs تتبع معايير:
- RESTful design
- JWT authentication
- Rate limiting
- Arabic + English support

## كيفية القراءة

كل مستند PRD يتضمن:

1. **نظرة عامة**: ملخص تنفيذي وأهمية الميزة
2. **المتطلبات الوظيفية**: تفاصيل كل وظيفة
3. **مخطط قاعدة البيانات**: جداول SQL الكاملة
4. **واجهات API**: نقاط النهاية المطلوبة
5. **التكامل مع KAIA**: الامتثال الشرعي
6. **مقاييس النجاح**: KPIs المستهدفة
7. **خطة التنفيذ**: المراحل والجدول الزمني
8. **تقدير التكاليف**: التطوير والتشغيل
