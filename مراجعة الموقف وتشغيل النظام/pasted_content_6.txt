15 دقيقة تثبت لنا “Week 1 Complete” فعلاً (Proof Checklist)

نفّذ دول من Root المشروع (على نفس الماك اللي عليه الكود):

1) إثبات ملفات النشر موجودة

ls -la Dockerfile docker-compose.yml 2>/dev/null || echo "Missing Docker files"

2) تشغيل Docker فعليًا

docker compose up -d --build
docker compose ps
docker compose logs -n 80 app
docker compose logs -n 80 db

النجاح = app running + db healthy + مفيش crash loop.

3) إثبات إن التطبيق بيرد على Port 8080

curl -i http://localhost:8080/ | head -30

4) إثبات إن DB شغالة وبتستقبل اتصال

docker exec -it $(docker ps -q --filter "name=db") psql -U haderos -d haderos_prod -c "SELECT 1;"

5) إثبات إن شاشات الـ UI موجودة في الكود

(غيّر المسارات حسب مكان ملفات React عندك)

grep -R "ERPDashboard" -n apps | head
grep -R "CreateInvoice" -n apps | head
grep -R "ProductManagement" -n apps | head
grep -R "LiveShoppingPanel" -n apps | head

لو طلّعتلي نتائج البنود (2) و(3) و(4) بس، أقدر أقولك بدقة: Week 1 فعلاً مكتمل ولا ده تقرير وصفي.
