## üìä System Overview

### Key Features
- **E-Commerce Management:** Products, orders, inventory
- **HR System:** Employee management with document verification
- **Shipment Tracking:** Multi-company shipment management
- **Monthly Accounts:** Secure temporary employee accounts
- **OTP Verification:** Hybrid Email/SMS system
- **AI-Powered:** Document extraction and verification

### Technology Stack
- **Frontend:** React 19 + TypeScript + Tailwind CSS
- **Backend:** Express + tRPC + Drizzle ORM
- **Database:** MySQL
- **Storage:** S3-compatible
- **Authentication:** Manus OAuth + JWT
