What It Does
HaderOS is a pre-MVP platform that aims to combine:

AI/ML for predictive analytics and autonomous planning
Islamic Finance compliance (Sharia-compliant operations via KAIA Theology Engine)
Digital Governance with autonomous agents
Blockchain Integration (ERC-3643 smart contracts on Ethereum/Polygon)
Current Status: Extensive architecture planning and documentation (16MB of specs), but minimal actual implementation code.

Tech Stack
Backend:

FastAPI 0.104.1 + Uvicorn (async)
PostgreSQL 15+ (SQLAlchemy ORM)
Redis (caching), Kafka + RabbitMQ (messaging)
TensorFlow 2.15, PyTorch 2.1 (ML/AI)
Web3.py 6.11 (blockchain)
JWT auth, Prometheus monitoring
Frontend:

React 18.2 + TypeScript 5.3
Vite 5.0, TailwindCSS 3.3
Zustand (state), React Router v6
ethers.js 6.10 (Web3)
DevOps:

Docker + Kubernetes ready
Code quality: Black, Pylint, ESLint, Prettier
Testing: pytest, Vitest
Main Entry Points
Backend:

Entry: backend.api.v1.main:app
Run: ./run_backend.sh
Port: 8080
Command: python -m uvicorn backend.api.v1.main:app --host 127.0.0.1 --port 8080 --reload

Frontend:

Entry: frontend/dashboard/src (React app)
Dev: npm run dev
Port: 3000

Quick Start:

make install    # Install all dependencies
make dev        # Start development environment
./run_backend.sh # Start backend server
